# Security

## 100% Secure Layer
การพัฒนา Mini App ในทางรัฐ จำเป็นต้องอยู่บน SSL โดยทางรัฐไม่รองรับการใช้งานบน http protocol หากมี request ที่เป็น http จะถูก Refuse โดยอัตโนมัติ

## Log การใช้งาน
ทางรัฐจะไม่มีการเก็บ Sensitive Data (ข้อมูลหน้าบัตรประชาชน) ไว้บน Server ของทางรัฐ จะเก็บเพียงข้อมูลประวัติการใช้งาน และขอเข้าถึงบริการเท่านั้น
